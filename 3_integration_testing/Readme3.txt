Testing Rights:
Log in as an admin user (See README1.txt for login info).
See if Add, Edit, and Delete icons are clickable.

Log out.

Log in as user without edit rights.
See if Add, Edit, and Delete icons are clickable.


Testing successful search:
Type any field from a known existing record into the search bar and click the magnifying glass icon.
The list should be shortened to only matching entries.
Click return home to retrieve the full list.

Testing failed search:
Type random characters into the search bar that do not match any records and click the magnifying glass icon.
Should present an error that says "No results! Your search returned no results, or an error has occurred. Click here to return to the homepage."
Clicking the blue link that says "Click here to return to the homepage." returns you to the entire list.
